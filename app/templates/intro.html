Data-Driven Documents (D3)

What is D3?

  • JavaScript library for data visualization and manipulation
  • It is not a chart library (but it can make charts)
  • Uses HTML, CSS, and SVG to render visualizations on web browsers

HyperText Markup Language (HTML)

  • Web standard for creating web pages

CSS & SVG

  • Cascading Style Sheets (CSS) helps describe the presentation of a document. Applies to HTML, XML, and SVG
  • CSS is designed to separate the presentation details (fonts, sizes, colors) from the content
  • Scalable Vector Graphics (SVG) is an image format for two-dimensional graphics
  • Based on XML, so it can be compressed, indexed, searched, and scripted

Document Object Model

  • An application programming interface that maps HTML/XML to a tree structure
  • The DOM is a web standard, cross-platform and language independent
  • Used to programatically modify and render documents from the tree structure

D3 is Data ↦ Elements

  • Visualizing Data with Web Standards (HTML/SVG)
  • Data ↦ Elements – constructing the DOM from Data
  • In visualization, each data point has a corresponding element (graphical marks). D3 helps you maintain this mapping!
